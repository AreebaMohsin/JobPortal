# Online Job Portal Project
Online Job Portal project is web application built using PHP, MySQL as backend and HTML JavaScript , Bootstrap as Frontend technologies. <br/>
This platform serves as an online marketplace connecting job-seekers with employers. Job-seekers create profiles (with resumes, skills, experience, preferred location, etc.), and employers post job openings.
